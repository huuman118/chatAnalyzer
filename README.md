# Welcome to Huu's Slack Channel Analyzer

This app automatically:

* scrapes messages from the FullStack Slack Channel #huuchatappchannel
* sends the data to Watson for tone analysis
* displays the results via this app

The app uses the following techonologies:

* Slack API
* IBM Watson's Tone Analyzer API
* Material-UI
* d3 via Nivo Package

visit the deployed site at https://huuman-chatanalyzer.herokuapp.com/
